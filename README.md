 Landing Page

Landing Page desenvolvida para o e-book “O Guia Essencial: Para Cabelos Saudáveis em Casa”, um produto digital voltado para mulheres que desejam recuperar a saúde dos fios sem depender de salão.

O projeto foi construído com:

✅ HTML5

✅ CSS3 (layout responsivo e estética premium)

✅ JavaScript (interações e comportamento da página)

🎯 Objetivo do Projeto

Criar uma página com forte apelo emocional e visual sofisticado, transmitindo:

Sensação de exclusividade e cuidado

Jornada de transformação (dor → processo → resultado)

Clareza na proposta de valor

Estrutura estratégica para conversão

🎨 Conceito Visual

Design escuro com contraste elegante, tipografia refinada e composição focada em destacar o produto digital (mockups em dispositivos), reforçando autoridade e valor percebido.
